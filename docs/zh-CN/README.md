# 图标替换工具

这是一个面向 Windows 的桌面工具，用来把自定义图标应用到文件夹、快捷方式和 EXE 派生快捷方式上。

## 支持范围

- 目标对象支持：
  - 文件夹
  - 快捷方式 `.lnk`
  - `exe` 程序
- 图标来源支持：
  - `.png`
  - `.jpg`
  - `.jpeg`
  - `.ico`
  - `.exe`

## 行为说明

- 目标是文件夹时：
  - 程序会写入 `desktop.ini`
  - 会给文件夹设置系统属性和只读属性
  - 不会改动文件夹内部原有文件
- 目标是 `.lnk` 时：
  - 直接修改快捷方式的图标字段
  - 不会修改快捷方式指向的原始文件
- 目标是 `exe` 时：
  - 不会修改 `exe` 本体
  - 会在同目录下生成 `原文件名 - Custom Icon.lnk`

## 图标处理逻辑

- 图片来源会先被裁剪为正方形，再导出为多尺寸 `.ico`
- 已有 `.ico` 会重新标准化尺寸
- `exe` 来源会直接解析资源表并提取最佳图标
- 所有最终使用到的图标文件都会存放到 `%LOCALAPPDATA%\IconConversionTool\icons`

## 运行方式

```powershell
pip install -r requirements.txt
python app.py
```

## 双语支持

- 界面语言文件位于 `lang/`
- 当前内置：
  - `lang/zh-CN.json`
  - `lang/en-US.json`
- 程序启动后可以在右上角切换语言
- 语言选择会保存到项目根目录下的 `.icon-tool-settings.json`

## 后续扩展建议

- 如果你想继续增加语言，只需要新增一个 JSON 文件并保持同样的键结构
- 如果想把提示文案、错误文案和按钮文字继续细分，现在这套结构已经可以继续扩展
